# Markdown 入门教程：轻松写好博客内容
欢迎来到我的 Markdown 学习笔记，这篇文章会教你快速掌握博客必备的 Markdown 语法。

## 一、基础语法总结
### 1.1 标题与段落
- 标题用 `#` 表示，最多 6 级
- 段落之间空一行即可分隔

### 1.2 列表与强调
1.  有序列表用「数字 + 句点」表示
2.  无序列表用 `-` 表示
3.  重点内容用 **粗体** 或 *斜体* 强调
